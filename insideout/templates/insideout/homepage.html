Inside Out

This page is defined by a template at insideout/templates/insideout/homepage.html. You can change it. The github repository for this project is currently at https://github.com/cproctor/insideout.

Routes

  • /: Homepage.
  • /people/new/<name>: Create a new person.
  • /people/<pid>: Show a person.
  • /people/<pid>/set/<attribute>/<value>: Set a person's attribute.
  • /classrooms/new/<name>: Create a new classroom.
  • /classrooms/<cid>: Show a classroom.
  • /people/<pid>/classroom/<cid>/join: Add a person to a classroom.
